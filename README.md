# 🏥 Sushruta Hospital – Medical Management Platform

A high-performance, bilingual (English/Telugu) hospital management system designed for **Sushruta Hospital**. This platform features a premium user interface for patients to book appointments and a robust admin dashboard for hospital staff to manage daily operations.

## ✨ Key Features
- **Bilingual Support**: Full English and Telugu localization with optimized typography (Noto Sans Telugu).
- **Online Appointment Booking**: Seamless form integration with Supabase and real-time backend notifications.
- **Admin Dashboard**: Secure management suite featuring:
  - Real-time booking list with Search & Filter.
  - One-click WhatsApp patient messaging.
  - Data export to CSV/Excel.
  - Statistical insights for hospital performance.
- **Modern UI/UX**: Built with Playfair Display and Plus Jakarta Sans for a premium medical aesthetic.
- **SEO Optimized**: Advanced meta tags, OpenGraph previews, and structured JSON-LD data for local search dominance.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+), FontAwesome.
- **Backend**: Node.js, Express.js.
- **Database**: Supabase (PostgreSQL).
- **Deployment**: Vercel (Frontend), Render (Backend).

## 🚀 Deployment Instructions
The frontend is pre-configured for **Vercel** with a `vercel.json` file. Simply drag the `frontend` folder into your Vercel dashboard to go live.
